# Portfolio Cao Huy Hưng

Trang web Portfolio cá nhân chuyên nghiệp, hiện đại được thiết kế theo phong cách **Glassmorphism sáng**. 

## Công nghệ sử dụng
- HTML5 & CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Google Fonts (Inter & Outfit)
- Material Symbols & Font Awesome Icons

## Hướng dẫn triển khai lên GitHub Pages

Để đưa trang web này lên mạng (deploy) bằng GitHub Pages, bạn hãy làm theo các bước sau:

1. **Khởi tạo và đẩy mã nguồn lên GitHub:**
   - Tạo một repository mới trên GitHub (ví dụ: `my-portfolio`).
   - Mở terminal trong thư mục này và chạy các lệnh:
     ```bash
     git init
     git add .
     git commit -m "Initial commit: Portfolio Cao Huy Hưng"
     git branch -M main
     git remote add origin https://github.com/USER_NAME/REPO_NAME.git
     git push -u origin main
     ```
   *(Thay `USER_NAME` và `REPO_NAME` bằng thông tin của bạn)*

2. **Kích hoạt GitHub Pages:**
   - Truy cập vào repository của bạn trên GitHub.
   - Chọn tab **Settings** (Cài đặt).
   - Chọn mục **Pages** ở thanh bên trái.
   - Trong phần **Build and deployment**, chọn Branch là `main` và folder là `/(root)`.
   - Nhấn **Save**.

3. **Xem kết quả:**
   - Sau vài phút, GitHub sẽ cung cấp cho bạn một đường dẫn (URL) dạng `https://USER_NAME.github.io/REPO_NAME/`.
   - Truy cập đường dẫn đó để xem trang web của bạn đã online!
